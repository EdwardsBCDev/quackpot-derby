Family Game Night

Quackpot Derby

One big host screen. Twenty funky ducks. Phone betting by QR code. Big random races with changing leaders and a proper finish.

Open Host Screen Join As Player

How It Works

1. Host opens the TV screen and gets a room QR code. 2. Players scan, enter a name, and receive £10. 3. Host opens betting for Race 1. 4. Players pick a duck and place a stake. 5. Host closes betting, ducks line up, then the race runs.

Design Direction

This rebuild is focused on a two-screen party setup rather than a single-page control panel. The host gets the show. Players get the fastest possible join-and-bet flow.
